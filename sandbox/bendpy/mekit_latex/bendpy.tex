\abstract{
We present a light-weight Python module for computing small deformations of a 
single plate supported by an arbitrary number of possibly intersecting
stiffeners,
whose shape need not to be linear. We show how the problem fits into the
framework 
of abstract saddle point problems and how this abstraction can be exploited for
clean design of the code. We discuss properties of the resulting linear system
for two different sets of basis functions, namely, the eigenfunctions of the 
biharmonic operator and specialized Legendre polynomials.
}
